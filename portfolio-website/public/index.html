Weather Monitoring System

I recently developed a real-time Weather Monitoring System that integrates with the OpenWeatherMap API to provide real-time weather updates for major Indian cities like Delhi, Mumbai, and Bangalore. This project includes:

  • 1. Continuous weather data retrieval and analysis
  • 2. Daily summaries with temperature rollups (average, max, min) and dominant weather condition
  • 3. User-configurable threshold alerts for extreme conditions
  • 4. Data visualization for historical trends and daily summaries
  • 5. Built using Python, Flask, Matplotlib, and other modern tools

    • The system demonstrates skills in API integration, data processing, visualization, and alert systems, making it a great example of functional and scalable application design.

    Dynamic Rule-Based Eligibility Engine with AST Representation

    Developed a 3-tier application to dynamically evaluate user eligibility based on customizable rules using Abstract Syntax Trees (AST). The system includes a user-friendly web interface (HTML/CSS/JavaScript), a Flask-based API backend, and SQLite for data persistence. Users can define, modify, and combine rules like (age > 30 AND department = 'Sales') OR (salary > 50000) to determine eligibility dynamically.
